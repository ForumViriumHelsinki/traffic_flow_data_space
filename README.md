# Traffic Flow Data Space

Repository for the Traffic Flow Data Space (TFDS) projects data space design and development. This project is being carried out in collaboration with Forum Virium Helsinki, Porto Digital, and Nationaal Dataportaal Wegverkeer (NDW).

## Traffic Flow Data Space architectural design and development

The data space architectural design is developed using [mermaid](https://github.com/mermaid-js/mermaid).
