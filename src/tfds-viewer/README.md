# TFDS / utils / tfds-viewer

This directory contains the **TFDS Data Products Demonstrator**, a web-based utility for visualizing various traffic data products generated by the TFDS processes.

## Overview

The demonstrator is a single-page application built with HTML, CSS, and JavaScript, utilizing the [Leaflet](https://leafletjs.com/) library for interactive mapping. It provides a visual representation of traffic disturbances, speed warnings, and roadwork impacts in the Helsinki region.

## Components

### Web Application
* **index.html** - The main entry point for the viewer. It initializes the map, fetches data products, and renders them with informative popups and legends.

### Data Samples
These files are local samples of the data products visualized by the viewer:
* **datex2_output.json** - A DATEX II `SituationPublication` containing road statuses (open/closed) and work area polygons.
* **filtered_segments.json** - A GeoJSON collection of road segments where real-time speeds are significantly lower than typical speeds.
* **traffic_disturbance_max_running_mean.json** - A GeoJSON collection of traffic disturbance center points, weighted by their impact on historical traffic flow.

*Note: The application is configured to fetch the latest live data from Google Cloud Storage (`https://storage.googleapis.com/tfds-public-output/`).*

## Usage

To view the demonstrator, open `index.html` in any modern web browser. 

The map features several interactive layers:
1.  **DATEX II Data**: Shows road closures (red), open roads (green), and work areas (blue polygons).
2.  **Slow moving segments**: Highlights areas with significantly reduced speeds.
3.  **Roadworks**: Displays the impact of roadworks on traffic flow with color-coded intensity (Very Low to Very High).

A legend in the bottom-right corner provides details on the symbology used for each layer.
