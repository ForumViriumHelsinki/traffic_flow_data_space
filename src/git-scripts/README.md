# TFDS Helper Scripts

This folder contains shell script utilities for cloning and updating relevant repositories in the Traffic Flow Data Space local development workspace.

## Scripts

*   **[simpl_cloner.sh](simpl_cloner.sh)**: Automatically clones and sets up designated SIMPL repositories for data space developments.
*   **[git_puller.sh](git_puller.sh)**: Performs git pull operations to synchronize all checked-out codebases with their upstreams.
